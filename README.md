# WordMax
Всем привет. Я так же как и вы устали от белых списков в рф? и вы просто хотите общаться с людьми без головной боли?
я создал на основе Яндекс документов а точнее word файла сервер, который проверяет последнее изменение файла, и присылает уведомление в телеграм (да он заблокирован. Но те у кого нету блокировок могут увидить то что вы написали им.)

# Что понадобиться ?
1. Получить Яндекс OAuth-токен можно здесь: https://yandex.ru/dev/disk/poligon/ → кнопка "Получить OAuth-токен"
2. Телеграм токен. И ваш пользовательский PeerID, для того чтобы уведомление не шли всем.
3. Зависимости:
   3.1. axios
   3.2. dotenv
4. и настроить .env файл.
5. готово!
